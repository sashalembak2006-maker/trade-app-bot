Технічний аналіз — стислий курс

Тренд + структура (крипто приклад ETH)

    HH ─────●  Higher High (висхідний тренд)
           ╱
    HL ───●    Higher Low
         ╱
    LL ●       Пробій HL вниз = можлива зміна тренду → PUT

RSI + рівні

RSI < 30 біля підтримки → зона для CALL. RSI > 70 біля опору → зона для PUT.

Патерн «Прапор» на M5

    │ Імпульс вгору
    ├─┐─┐─┐  Консолідація (прапор)
    └─┴─┴─┘
    Пробій вгору + обсяг → CALL на 1–3 хв експірації

Три екрани (адаптація Elder)

  1. H1 — напрямок тренду
  2. M15 — сигнал (MACD / RSI)
  3. M1 — точний вхід у Mini App
